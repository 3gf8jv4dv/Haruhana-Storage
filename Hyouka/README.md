<p align="center">
	斯人已逝，精神长存；沉舟侧畔，千帆竞发；<br>
	你们灵魂中的迸发的火花，已然闪现在我等生命的章节里，层层叠叠…<br>
	纵然你们故去，我等仍将永远铭记…<br>
</p>
<p align="center">
	此致<br>
	京都动画殉难者
